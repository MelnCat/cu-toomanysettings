# TooManySettings

> [GitHub Repository](https://github.com/MelnCat/cu-toomanysettings)

Adds more run settings to the game that may or may not be balanced.

Warning: Turning someone of the settings too high may result in lag, 
although you should probably expect for turning something to 100x.

# Settings

## Minibarrel Multiplier

The amount of minibarrels to spawn. Defaults to `1x`. 

Turn this higher if you want more minibarrels everywhere, or lower if you hate fun and gambling.

## Enemy Multiplier

The amount of enemies to spawn. Defaults to `1x`.

This can be useful if the base trap multiplier is increased, as you can use this setting to decrease enemies specifically while keeping nonliving traps still high.

## Geyser Multiplier

The amount of geysers to spawn. Defaults to `1x`.

Turn this up if you want to suffer.

## Plant Multiplier

The amount of plants (like glowplants) to spawn. Defaults to `1x`.

## Fluid Multiplier

The relative size of naturally spawning fluid pools to spawn (like lakes and stuff). Defaults to `1x`.

## Climbable Multiplier

The amount of climbable rope/vine to spawn. Defaults to `1x`.

## Spawn Extra Random Loot

Whether to spawn random items around the world. Defaults to `false`.

This is NOT the random items spawned on corpses. This is a separate custom spawning mechanism.

## Extra Random Loot Multiplier

The amount of random items to spawn for the Spawn Extra Random Loot setting. Defaults to `1x`.

Does nothing if Spawn Extra Random Loot is off.
